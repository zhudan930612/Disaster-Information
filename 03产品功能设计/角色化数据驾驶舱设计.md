## 功能单元 1：角色化数据驾驶舱

> 合并说明：本功能单元完成后，可将“功能点描述”章节合并到 [统计分析与管理驾驶舱.md](./统计分析与管理驾驶舱.md) 中。

### 功能概述

在现有统计模块只支持“通用统计页”的基础上，新增一组按角色分层的数据驾驶舱页面。不同角色进入同一菜单后，系统根据角色自动进入对应的数据首页：业务运营看业务数据 + 练习数据 + 竞赛数据，省级租户和县级角色只看权限内练习与竞赛数据，国家级角色看全国总体与各省明细。页面以指标卡、趋势图、排行图、分布图和明细表为主要呈现方式。

---

### 业务流程

#### 流程图
```text
用户登录后台
   ->
进入「数据驾驶舱」
   ->
系统识别角色
   ->
按角色加载默认统计范围
   ->
进入对应首页
   ->
查看指标卡 / 图表 / 明细表
   ->
调整筛选条件
   ->
刷新当前页数据
   ->
点击图表下钻或导出
```

#### 业务规则

**触发条件**
- 用户拥有`数据驾驶舱`菜单权限。
- 用户进入后台后点击`数据驾驶舱`菜单。

**处理逻辑**
1. 系统根据当前账号角色识别用户属于`业务运营角色`、`省级租户角色`、`县级角色`或`国家级角色`。
2. 系统根据角色绑定默认可见范围：
   - 业务运营：全平台业务数据、练习数据、竞赛数据
   - 省级租户：本省权限内练习数据、竞赛数据
   - 县级角色：本县权限内练习数据、竞赛数据
   - 国家级角色：全国赛事总体数据及各省明细数据
3. 页面初次加载时默认取最近 30 天数据；如果用户选择赛事或时间范围，则按筛选结果刷新。
4. 页面指标卡、图表和明细表使用同一组筛选条件口径。
5. 页面支持按角色范围导出当前明细结果，但不允许导出超越权限范围的数据。

**边界情况**
- 无数据时，指标卡显示 `0`，图表显示空状态，明细表显示“暂无数据”。
- 培训平台尚未同步到竞赛平台的数据，不纳入统计。
- 服务到期省份不允许新增正式竞赛数据，但历史数据仍可查看。
- 国家级角色默认只读，无配置、删除、修正能力。

### 原型设计

#### 最终方案

**方案：同一菜单，按角色自动进入对应首页**

```
后台
┌────────────────────────────────────────────┐
│ 数据驾驶舱                                 │
├────────────────────────────────────────────┤
│ 系统根据角色自动进入对应首页               │
│ 业务运营 -> 业务运营数据页                 │
│ 省级租户 -> 省级数据页                     │
│ 县级角色 -> 县级数据页                     │
│ 国家级角色 -> 国家级数据页                 │
└────────────────────────────────────────────┘
```

说明：权限最清晰，页面最聚焦，开发上也容易把角色和默认筛选范围绑定，符合本项目“分级治理、角色分层、权限隔离”的原则。

**交互说明**

| 元素 | 交互行为 | 备注 |
|------|----------|------|
| 指标卡 | 默认展示当前值，悬浮显示口径说明 | 所有指标卡带图标 |
| 图表 | 支持悬浮查看 tooltip，点击图形可联动下方明细表 | 一期不强制支持跳转外部页 |
| 筛选条件 | 切换后点击`查询`统一刷新全页 | 默认最近 30 天 |
| 导出按钮 | 导出当前明细表结果 | 遵循角色权限 |
| 明细表 | 支持分页、排序 | 默认按核心指标降序 |

**页面状态说明**
- 默认状态：展示当前角色默认可见范围的数据
- 加载中：指标卡使用骨架屏，图表显示加载状态
- 空数据：指标卡显示 0，图表显示空状态，表格显示“暂无数据”
- 错误状态：页面顶部提示“数据加载失败，请稍后重试”

#### 角色自动分发

**入口路径**：
- 后台 -> 数据驾驶舱

**页面名称**：无独立页面，作为菜单进入逻辑

**页面功能**：
1. 系统识别当前账号角色
2. 自动跳转到对应数据首页
3. 绑定默认筛选范围和默认指标集

**业务规则**：
- 一个账号仅进入一个默认首页。
- 如账号同时具备多个角色，以优先级进入：国家级 > 业务运营 > 省级 > 县级。
- 后续如需支持手工切换其他视图，必须仅允许切换到当前账号已授权视图。

#### 业务运营数据驾驶舱

**入口路径**：
- 后台 -> 数据驾驶舱 -> 业务运营数据页

**页面名称**：“业务运营数据驾驶舱”页面

**页面功能**：

1. **筛选区**
   - 日期范围：默认近 30 天
   - 省份：默认全部
   - 服务状态：全部 / 服务中 / 即将到期 / 已到期
   - 赛事：全部赛事

2. **核心指标卡**
   - `已开通省份数` `[图标：building]`
   - `服务中省份数` `[图标：shield]`
   - `即将到期省份数` `[图标：clock]`
   - `累计练习用户数` `[图标：users]`
   - `昨日练习人次` `[图标：pencil]`
   - `累计参赛用户数` `[图标：trophy]`
   - `昨日参赛人次` `[图标：chart-line]`
   - `近30天活跃省份数` `[图标：fire]`

3. **图表区**
   - 省份服务状态环图
   - 近 30 天练习/竞赛趋势双折线图
   - 省份活跃排行柱状图
   - 练习到竞赛转化漏斗图

4. **明细表**
   - 字段：省份名称、服务状态、累计练习用户数、累计参赛用户数、昨日活跃人数、最近更新时间、服务结束日期

5. **操作按钮**
   - `查询`
   - `导出`

**业务规则**：
- 业务运营页可见全平台数据，但以汇总和省份维度明细为主，不展示个人级明细。
- `即将到期省份数`默认指 30 天内到期的省份数。
- `近30天活跃省份数`按近 30 天内有练习或竞赛行为的省份去重统计。
- 漏斗图按“已同步用户 -> 有练习用户 -> 有参赛用户 -> 有晋级用户”统计。

#### 通用导出与下钻

**触发操作**：
- 点击任一角色页中的`导出`
- 点击图表中的某一柱、某一省份或某一漏斗层级

**页面名称**：“驾驶舱明细联动”能力

**页面功能**：

1. **导出当前结果**
   - 导出当前明细表结果
   - 文件命名：`角色名称_页面名称_YYYYMMDDHHmmss.xlsx`

2. **图表联动**
   - 点击图表数据点后，自动更新下方明细表筛选条件
   - 明细表标题同步更新当前联动条件

3. **重置联动**
   - 支持清空图表联动筛选，恢复页面原始视图

**业务规则**：
- 图表联动只作用于当前页，不跨页跳转。
- 导出范围遵循图表联动后的当前结果。
- 国家级、业务运营角色可导出全国或全平台汇总明细；省级和县级角色只能导出本级权限范围。

### 验收标准

- [ ] 用户进入`数据驾驶舱`菜单后，系统能根据角色自动进入对应首页。
- [ ] 业务运营角色首页同时展示业务数据、练习数据和竞赛数据。
- [ ] 省级租户角色首页仅展示本省权限内的练习数据和竞赛数据。
- [ ] 县级角色首页仅展示本县权限内的练习数据和竞赛数据。
- [ ] 国家级角色首页可查看全国总体数据和各省明细数据，且页面为只读。
- [ ] 四类页面都包含指标卡、图表区、明细表和导出能力。
- [ ] 所有指标卡都带图标展示，且悬浮可查看口径说明。
- [ ] 所有图表都支持 hover 查看数据，支持点击后联动下方明细表。
- [ ] 所有统计结果都遵循当前角色的权限范围和筛选条件。
- [ ] 培训平台未同步到竞赛平台的数据，不计入任意角色页面的统计结果。
- [ ] 无数据时，页面显示 0 和空状态，不出现权限外报错。
- [ ] 服务到期省份的历史统计数据仍可在有权限的角色页中查看。
