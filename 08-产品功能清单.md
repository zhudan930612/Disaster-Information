# 灾害信息员大赛平台 · 产品功能清单

## 1. 文档定位

本文档承接《总体产品架构与核心业务流程》及各专项设计文档，将平台需要交付的能力，按照产品界面和模块维度拆分为可执行的功能清单。

本文档重点回答：

- 学员竞赛小程序要做哪些功能
- 竞赛管理后台要做哪些功能
- 平台运营配置后台要做哪些功能
- 各功能的优先级如何划分

本文档不展开每个功能的交互逻辑和字段规则，后续在页面级功能设计文档中逐一展开。

**优先级说明：**

- **P0**：核心链路，首批必须上线，缺少则主流程不成立
- **P1**：首批建设范围内，与 P0 并行推进
- **P2**：首批建设范围内，可在 P0/P1 稳定后推进

## 2. 学员竞赛小程序

### 2.1 登录与身份

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 从国家级平台跳转进入 | 接收跳转凭证，换取本平台登录态 | P0 |
| 首次进入建立身份映射 | 首次登录时自动完成身份关联，不要求用户手动操作 | P0 |
| 登录态保持与续期 | 会话有效期内免重复登录 | P0 |
| 跳转凭证失效兜底 | 凭证失效时提示用户重新从国家级平台进入，不白屏 | P0 |
| 身份映射失败兜底 | 映射失败时展示友好提示，记录异常日志 | P1 |

---

### 2.2 首页

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 练习赛快捷入口 | 首页直接展示练习赛入口，一键发起 | P0 |
| 当前可参与赛事展示 | 展示本省已发布的正式赛事（初赛/复赛/决赛），含状态和截止时间 | P0 |
| 剩余答题次数展示 | 展示基础次数和奖励次数各自剩余量 | P0 |
| 学习联动任务入口 | 次数不足或常态展示学习任务引导，点击跳转国家级平台 | P0 |
| 赛事公告/通知展示 | 展示管理员发布的赛事通知 | P1 |

---

### 2.3 练习模块

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 发起练习赛 | 消耗一次答题次数，系统按组题策略生成题目集合 | P0 |
| 答题过程 | 支持判断题、单选题、多选题三种题型展示与作答 | P0 |
| 计时展示 | 答题过程中实时展示已用时长 | P0 |
| 答题结果展示 | 答完后展示本次成绩、正确率、耗时 | P0 |
| 逐题解析展示 | 结果页展示每道题的正确答案和解析 | P0 |
| 次数不足拦截与引导 | 次数为 0 时拦截发起，提示前往国家级平台完成学习任务获取次数 | P0 |
| 奖励次数到账提示 | 国家级平台完成任务后返回，展示次数到账结果或待到账提示 | P0 |
| 练习历史列表 | 展示历史练习记录（时间、成绩、正确率） | P1 |
| 错题记录 | 记录用户答错的题目，支持查看 | P1 |
| 练习排行榜 | 展示本省/全平台练习成绩排行（快照） | P1 |

---

### 2.4 赛事模块

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 可参与赛事列表 | 展示本省已开通且用户有权参与的赛事 | P0 |
| 赛事详情页 | 展示赛事规则、阶段信息、时间、题量、时长、门槛说明 | P0 |
| 参赛资格校验 | 进入赛事前校验门槛（省份、在岗状态、练习次数、组织范围等），不通过时展示原因 | P0 |
| 个人赛答题 | 正式赛事个人赛答题流程，含计时、同卷机制 | P0 |
| 个人赛成绩查看 | 答完后展示本次成绩、排名、晋级状态 | P0 |
| 团体赛信息展示 | 展示所属团队名称、队员组成、团队排名 | P0 |
| 团体赛答题 | 以个人身份参与团体赛答题，成绩计入团队 | P0 |
| 阶段进展展示 | 展示初赛/复赛/决赛各阶段的状态（未开始/进行中/已结束/已晋级/未晋级） | P0 |
| 赛事排行榜 | 展示当前阶段个人排行或团队排行 | P1 |
| 赛事结果发布展示 | 阶段结束后展示官方发布的结果通知 | P1 |

| 个人排行查看 | 展示本人在练习赛/正式赛事中的排名位置 | P1 |

## 3. 竞赛管理后台

### 3.1 赛事管理（省级主管管理员）

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 创建赛事 | 新建正式赛事，选择赛事类型（个人赛/团体赛）、填写基本信息 | P0 |
| 赛事规则配置 | 按固定赛制配置各阶段参数：时间、题量、时长、晋级比例、题库范围、参赛门槛 | P0 |
| 发布赛事 | 将配置完成的赛事发布，触发题目集合生成与锁定 | P0 |
| 暂停/结束赛事 | 支持在必要时暂停进行中的赛事或提前结束 | P1 |
| 赛事阶段推进 | 初赛结束后触发晋级结算，开放复赛报名；以此类推 | P0 |
| 赛事结果发布 | 各阶段结束后由管理员确认并发布官方结果 | P0 |
| 赛事公告发送 | 创建并发送面向参赛学员的通知公告 | P1 |

---

### 3.2 报名与资格管理（省级主管管理员）

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 参赛名单查看 | 查看报名或具备参赛资格的学员列表，含所属组织和资格状态 | P0 |
| 资格审核 | 对参赛资格进行人工审核（如有需要）或查看系统自动校验结果 | P1 |
| 取消参赛资格 | 针对特殊情况取消某学员的参赛资格 | P1 |
| 团体赛分组管理 | 按组织维度查看团队分组情况，不支持手动跨组调整 | P0 |

---

### 3.3 统计与结果查看（省级主管管理员）

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 练习数据看板 | 本省练习赛活跃人数、练习次数、平均正确率等 | P1 |
| 赛事数据看板 | 参赛率、完赛率、成绩分布、阶段通过率 | P0 |
| 个人成绩排行 | 按阶段查看个人成绩排行列表 | P0 |
| 团队成绩排行 | 团体赛各团队成绩排行 | P0 |
| 晋级结果列表 | 各阶段晋级名单 | P0 |
| 学习联动效果统计 | 奖励次数发放总量、学习任务完成人数等 | P1 |
| 数据导出 | 导出成绩表、排名表、晋级名单等 | P1 |

---

### 3.4 名单与执行管理（地市县执行管理员）

| 功能 | 说明 | 优先级 |
|------|------|--------|

| 本级成绩结果查看 | 查看本级人员在赛事中的成绩与排名 | P0 |
| 赛事信息查看 | 查看已发布赛事的规则、时间、阶段说明 | P0 |

### 3.5 国家级管理员只读视图

| 功能 | 说明 | 优先级 |
|------|------|--------|

| 各省开通状态列表 | 查看各省租户开通状态和能力开通情况 | P1 |
| 各省赛事统计摘要 | 各省参赛率、成绩汇总等省级维度统计 | P1 |

## 4. 平台运营配置后台

### 4.1 租户管理

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 创建租户 | 新建省级租户，初始化基础信息 | P0 |
| 租户生命周期管理 | 切换租户状态（待开通/联调中/已上线/已到期/已关闭） | P0 |
| 能力开关配置 | 配置正式赛事、团体赛等能力是否对该租户开放 | P0 |
| 租户列表与状态总览 | 查看所有租户的开通状态和基础信息 | P0 |

---

### 4.2 组织与人员配置

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 组织架构配置 | 配置省/市/县/乡镇层级结构和组织树 | P0 |
| 人员数据查看与校验 | 查看从国家级平台同步的人员数据，处理异常映射 | P0 |
| 异常人员手工校正 | 对映射失败或归属错误的人员进行手工修正 | P1 |
| 角色分配 | 为管理员账号分配对应角色和权限范围 | P0 |

---

### 4.3 题库管理

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 共用题库题目管理 | 题目录入、编辑、启用/停用、审核 | P0 |
| 省级私有题库题目管理 | 代录或接收省级上传的私有题目，归属绑定 | P0 |
| 题库装载与绑定 | 将共用题库和省级私有题库装载至指定租户 | P0 |
| 题库版本管理 | 查看题目版本历史，管理历史快照 | P1 |
| 题库充足性查看 | 查看各题型、各难度等级的题目数量，评估是否充足 | P1 |

---

### 4.4 赛事机制参数配置

| 功能 | 说明 | 优先级 |

| 正式赛事参数边界配置 | 配置省级可调整参数的上下限（如题量范围、晋级比例范围等） | P0 |
| 答题次数基础规则配置 | 配置每日基础次数发放量 | P0 |

---

### 4.5 学习联动规则配置

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 学习任务映射配置 | 配置国家级平台哪些学习任务可触发次数奖励 | P0 |
| 次数奖励规则配置 | 配置单任务奖励次数、每日上限、累计上限、有效期规则 | P0 |
| 联动开关配置 | 开启/关闭学习联动能力 | P1 |

---

### 4.6 协同保障

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 协同日志查询 | 查看同步事件日志（同步时间、状态、失败原因） | P0 |
| 单用户同步重试 | 针对同步失败的用户手动触发重试 | P0 |
| 单用户奖励次数补发 | 针对未到账的奖励次数进行手工补发 | P0 |
| 比赛结果回流重试 | 比赛结果未成功回流国家级平台时手动触发重试 | P1 |

## 6. 当前未纳入功能

以下功能明确不在当前建设范围内：

- 本平台内课程学习与视频播放
- 自定义赛制引擎
- 高复杂度积分商城
- 社区交流/论坛
- 智能推荐与运营辅助
- 偏离竞赛主链路的扩展玩法
