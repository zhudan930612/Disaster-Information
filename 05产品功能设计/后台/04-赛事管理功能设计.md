## 2. 模块概述

赛事管理是竞赛管理后台的核心模块，支撑省级正式赛事从创建到结果发布的全生命周期。

**核心规则**：
- 赛事规则固定现有模板（初赛→复赛→决赛），省级管理员只能在参数边界内配置，不改模板底层逻辑
- 一个赛事只能关联一个本省自主题库
- 赛事发布后，系统从关联题库生成题目快照（同卷锁定），发布后不可更换题库
- 创建时若题库题目数量 < 赛事配置的题目数量，禁止发布，提示补充题目
- 运营/超级管理员代创建的竞赛，在省级后台显示"平台代运营"标识
- 市/县管理员无赛事管理操作权限

**赛事阶段定义**：

| 阶段 | 说明 | 晋级逻辑 |
|------|------|---------|
| 初赛 | 第一阶段，所有符合资格的参赛者均可参加 | 按排名或分数线晋级复赛 |
| 复赛 | 第二阶段，初赛晋级者参加 | 按排名或分数线晋级决赛 |
| 决赛 | 第三阶段，复赛晋级者参加 | 产生最终排名和奖项 |

省级管理员在创建赛事时，可选择启用哪些阶段（至少启用初赛）。

## 5. 赛事状态流转

### 5.1 状态定义

```
┌─────────┐    创建    ┌─────────┐    发布    ┌─────────┐    开始    ┌─────────┐
│  无    │ ────────> │ 未发布  │ ────────> │ 报名中  │ ────────> │ 进行中  │
└─────────┘          └─────────┘          └─────────┘          └─────────┘
                          │                   │                   │
                          │ 编辑              │ 取消报名          │ 阶段结束
                          ▼                   ▼                   ▼
                     ┌─────────┐         ┌─────────┐        ┌─────────┐
                     │ 已编辑  │         │ 报名截止 │        │ 已结束  │
                     └─────────┘         └─────────┘        └─────────┘
                          │                                        │
                          │ 发布                                   │ 发布结果
                          ▼                                        ▼
                     ┌─────────┐                              ┌─────────┐
                     │ 报名中  │                              │ 结果已发布│
                     └─────────┘                              └─────────┘
```

### 5.2 状态说明

| 状态 | 说明 | 可执行操作 |
|------|------|-----------|
| 未发布 | 赛事已创建，尚未对外发布 | 编辑、发布、删除 |
| 报名中 | 赛事已发布，处于报名开放阶段 | 查看报名、编辑（仅时间调整）、下架 |
| 报名截止 | 报名时间已过，等待比赛开始 | 开始赛事（手动/自动） |
| 进行中 | 赛事至少一个阶段正在进行 | 查看实时数据、强制结束 |
| 已结束 | 所有阶段已结束 | 发布结果、查看结果 |
| 结果已发布 | 成绩和排名已对外公布 | 查看结果、导出 |
| 已下架 | 运营强制下架或省级主动下架 | 仅查看，不可恢复 |

### 5.3 状态转换规则

| 转换 | 触发条件 | 校验规则 |
|------|---------|---------|
| 未发布 → 报名中 | 点击"发布" | 题库充足性校验通过；至少启用一个阶段；各阶段时间逻辑正确（开始时间 < 结束时间，阶段间不重叠） |
| 报名中 → 报名截止 | 到达报名截止时间 | 自动转换，无需人工操作 |
| 报名截止 → 进行中 | 到达阶段开始时间 | 自动转换；或管理员手动"开始" |
| 进行中 → 已结束 | 最后一个阶段结束时间到达 | 自动转换；或管理员手动"结束" |
| 已结束 → 结果已发布 | 管理员点击"发布结果" | 成绩计算完成、排名已生成 |
| 报名中 → 已下架 | 管理员点击"下架" | 二次确认；已报名人员收到下架通知 |
| 已下架 → 任何状态 | — | 不可恢复 |

## 7. 竞赛管理后台：赛事创建/编辑页

**页面路径**：`/province/competition/form?id={competitionId}`
**页面标题**：新建赛事 / 编辑赛事

### 表单区块

#### 7.1 基础信息

| 字段 | 字段名 | 类型 | 必填 | 校验规则 | 说明 |
|------|--------|------|------|---------|------|
| 赛事名称 | name | 文本 | 是 | 长度 5-100 | — |
| 赛事类型 | type | 单选 | 是 | — | 个人赛 / 团体赛 |

#### 7.2 阶段配置

阶段配置为动态表单，省级管理员可增删阶段，但至少保留一个阶段。

**阶段配置卡片（每个阶段一个卡片）**：

| 字段 | 字段名 | 类型 | 必填 | 校验规则 | 说明 |
|------|--------|------|------|---------|------|
| 阶段类型 | stageType | 单选 | 是 | 不可重复选择同一阶段类型 | 初赛 / 复赛 / 决赛 |
| 开始时间 | startTime | 日期时间 | 是 | 必须 >= 当前时间 | — |
| 结束时间 | endTime | 日期时间 | 是 | 必须 > 开始时间 | — |
| 答题时长 | duration | 数字 | 是 | 5-180 分钟 | 参赛者进入答题后的倒计时 |
| 题目数量 | questionCount | 数字 | 是 | 1-200 | 该阶段从题库中抽取的题目数 |
| 晋级方式 | promotionType | 单选 | 是 | 决赛无需配置 | 按排名晋级 / 按分数线晋级 |
| 晋级条件 | promotionValue | 数字 | 条件必填 | 排名：1-10000；分数线：0-100 | 如"前 100 名"或"80 分" |

**阶段顺序规则**：
- 阶段必须按初赛 → 复赛 → 决赛的顺序排列
- 前一阶段的结束时间必须 <= 后一阶段的开始时间（允许间隔）
- 仅启用初赛时，不展示复赛和决赛卡片

**阶段增删交互**：
- 默认展示"初赛"卡片
- 点击"添加阶段"，按顺序添加复赛/决赛卡片
- 点击卡片右上角"删除"可移除该阶段（至少保留一个）

#### 7.3 题库绑定

| 字段 | 字段名 | 类型 | 必填 | 校验规则 | 说明 |
|------|--------|------|------|---------|------|
| 关联题库 | bankId | 下拉选择 | 是 | — | 本省自主题库列表 |

**题库充足性校验**：
- 选择题库后，实时显示"该题库当前共 X 题"
- 若题库题目数 < 所有启用阶段的题目数量之和，显示红色警告："题库题目数量不足，当前 X 题，需要 Y 题（各阶段题目数之和），请补充题目后再发布。"
- 保存时若充足性校验未通过，可保存为"未发布"状态，但禁止"发布"

**题库更换规则**：
- 未发布状态下：可随时更换关联题库
- 已发布状态下：关联题库字段置灰不可编辑

#### 7.4 参赛范围（个人赛）

| 字段 | 字段名 | 类型 | 必填 | 说明 |
|------|--------|------|------|------|
| 参赛范围 | scope | 单选 | 是 | 全省开放 / 指定地区 |

**指定地区模式**：
- 展示地区选择器（基于机构树：省-市-县）
- 支持多选市/县级地区
- 不选表示该层级全部开放

#### 7.5 参赛范围（团体赛）

团体赛的参赛单位为县/市级，与个人赛不同。

| 字段 | 字段名 | 类型 | 必填 | 说明 |
|------|--------|------|------|------|
| 组队层级 | teamLevel | 单选 | 是 | 按县组队 / 按市组队 |
| 报名截止时间 | registerDeadline | 日期时间 | 是 | 团队赛需要在后台报名后方可参赛 |

**团体赛报名规则**：
- 团队赛默认不在小程序端开放自由报名
- 由省级/地市县管理员在后台维护参赛团队名单
- 报名截止后，系统根据报名名单确认参赛资格

### 页面操作按钮

| 按钮 | 触发条件 | 操作结果 | 成功提示 | 失败提示 |
|------|---------|---------|---------|---------|
| 保存 | 表单校验通过 | 保存为"未发布"状态，返回列表页 | 保存成功 | 显示具体错误 |
| 保存并发布 | 表单校验通过 + 题库充足性通过 | 保存并直接发布为"报名中"状态 | 发布成功 | 显示具体错误 |
| 取消 | 任意时刻 | 返回列表页，不保存 | — | — |

**校验规则汇总**：
- 赛事名称：必填，5-100字符
- 阶段时间：开始时间 >= 当前时间，结束时间 > 开始时间，阶段间不重叠
- 题目数量：每个阶段 >= 1
- 关联题库：必填
- 题库充足性：题库题目数 >= 各阶段题目数之和（仅发布时校验）

## 10. 竞赛管理后台：结果查看与导出

**页面路径**：`/province/competition/result?id={competitionId}`
**页面标题**：赛事结果

### 页面说明

结果页在赛事"已结束"或"结果已发布"状态下可查看。若赛事处于"进行中"状态，可查看实时答题数据但不展示最终排名。

### 10.1 阶段结果切换

页面顶部有阶段切换标签（初赛 / 复赛 / 决赛），点击切换查看不同阶段的结果。

### 10.2 成绩列表

**列定义**：

| 列名 | 字段 | 类型 | 说明 |
|------|------|------|------|
| 排名 | rank | 数字 | 按成绩排序 |
| 姓名 | userName | 文本 | — |
| 所属地区 | region | 文本 | — |
| 所属单位 | unit | 文本 | — |
| 成绩 | score | 数字 | 百分制 |
| 用时 | duration | 时间 | 答题耗时 |
| 晋级状态 | promotionStatus | 标签 | 已晋级 / 未晋级 / —（决赛无晋级） |

- 姓名：文本输入，模糊搜索
- 所属地区：下拉选择
- 晋级状态：下拉选择（全部/已晋级/未晋级）

**顶部统计栏**：
- 本阶段参赛人数：X 人
- 平均分：X 分

**操作按钮**：
- **导出当前阶段结果**
- **导出全部阶段结果**（多 Sheet，每个阶段一个 Sheet）

### 10.4 晋级名单

单独展示已晋级到下一阶段的名单，支持导出。

代运营创建竞赛的表单与省级后台的赛事创建页完全一致（见第 7 节），差异点：

## 13. 平台运营配置后台：竞赛强制下架

**操作入口**：全租户竞赛列表页的行内操作

### 强制下架弹窗

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 下架原因 | 文本域 | 是 | 长度 10-200 |

**影响范围**：
- 赛事状态立即变更为"已下架"
- 已报名用户收到下架通知（如对接通知服务）
- 已开始的赛事强制结束，已答题成绩保留但不再计入排名
- 省级后台该赛事显示"已下架"状态，不可恢复

**操作权限**：仅平台超级管理员可操作。
