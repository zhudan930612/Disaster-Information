## 2. 模块概述

国家级数据看板面向国家级管理员，提供全国层面的汇总数据查看能力。

**核心规则**：
- 国家级管理员仅查看，不参与赛事开通、配置和日常运营管理
- 数据范围为全国汇总 + 授权范围内部分租户级明细
- 所有数据为只读，无操作权限

## 4. 全国汇总看板

**页面路径**：`/national/statistics/overview`
**页面标题**：全国数据汇总

### 核心指标

| 指标 | 说明 |
|------|------|
| 累计参与人数 | 全国累计参与练习或竞赛的去重用户数 |
| 累计练习人次 | 全国累计练习次数 |
| 累计举办赛事数 | 全国各省级租户累计发布的正式赛事数 |
| 平均正确率 | 全国练习平均正确率 |

### 趋势图

- 全国练习参与趋势（近12个月折线图）
- 全国赛事举办趋势（近12个月折线图）
