# 学员竞赛小程序 · 完整功能设计

## 1. 文档定位

本文档承接《00-产品功能清单》和《DESIGN-小程序设计规范》，将小程序端全部页面级功能与视觉规范深度对齐，输出可直接指导前端开发的完整功能设计。

**本文档重点回答：**
- 小程序包含哪些页面，页面间如何跳转
- 每个页面的功能区块、UI元素、视觉规范如何落地
- 每个页面的交互规则、状态流转、异常处理

**本文档不展开：**
- 接口字段定义与接口协议（由后端接口文档覆盖）
- 数据埋点设计（由独立埋点文档覆盖）
- 动画时序与精确数值曲线（由前端实现时参考设计规范微调）

- 《00-产品功能清单》

### 4.3 功能区块

#### 4.3.1 品牌区

- **位置**：页面顶部居中
- **Logo**：小程序品牌图标，`80px x 80px`，圆角 `16px`
- **标题**："灾害信息员竞赛平台"，`22px`，`font-weight: 600`，`#1D2129`
- **副标题**："使用国家级平台账号登录"，`14px`，`#4E5969`

#### 4.3.2 账号输入

- **标签**："国家级平台账号"，`14px`，`#4E5969`，标签位于输入框上方，间距 `8px`
- **输入框**：
  - 背景 `#FFFFFF`，边框 `1px solid #E5E6EB`，圆角 `8px`，高度 `48px`
  - 内边距 `12px 16px`
  - 占位文字："请输入国家级平台账号"，`14px`，`#86909C`
  - 聚焦态：边框变为 `1px solid #1677FF`，外发光 `0 0 0 3px rgba(22,119,255,0.15)`
- **内容校验**：不能为空，首尾去空格

#### 4.3.3 密码输入

- **标签**："密码"，`14px`，`#4E5969`
- **输入框**：同账号输入框样式，类型为密码（默认隐藏，右侧带眼睛图标可切换显示）
- **眼睛图标**：`20px`，`#86909C`，点击切换明文/密文
- **占位文字**："请输入密码"

#### 4.3.4 用户协议勾选

- **勾选框**：`18px` 方形复选框，未选中时边框 `1.5px solid #E5E6EB`，选中时背景 `#1677FF` 带白色对勾
- **文案**："我已阅读并同意"，`13px`，`#4E5969`
- **链接文字**："《用户协议》"、"《隐私政策》"，`13px`，`#1677FF`，可点击跳转至协议详情页（WebView 或独立页面）
- **链接间文案**："和"，`13px`，`#4E5969`

#### 4.3.5 登录按钮

- **样式**：主按钮（蓝色实心），全宽（减去 `32px` 页面边距）
- **文案**："登录"，`16px`，`font-weight: 600`
- **禁用态**：未勾选协议或账号/密码为空时，按钮置灰（背景 `#A0C5FF`），不可点击
- **按压态**：背景 `#0958D9`

### 4.4 交互规则

| 触发条件 | 响应 |
|---------|------|
| 点击登录按钮 | 校验账号密码非空且已勾选协议 → 调用登录接口 |
| 登录成功 | 存储 Token，同步用户基础信息，跳转至练习首页 `/pages/practice/index` |
| 登录失败 | Toast 提示具体错误（如"账号或密码错误"），密码框清空 |
| 首次登录 | 向国家级平台校验身份，同步姓名、省份、单位、在岗状态，创建本平台用户并建立身份映射 |
| 非首次登录 | 校验 Token 有效性，同步最新基础信息 |

### 4.5 状态与异常

| 状态 | 表现 |
|------|------|
| 加载中 | 按钮显示加载动画（白色圆圈），禁用点击 |
| 网络异常 | Toast "网络异常，请检查网络后重试" |
| 国家级平台服务不可用 | Toast "登录服务暂不可用，请稍后重试" |
| 账号被禁用 | Toast "账号已被禁用，请联系管理员" |

#### 5.1.3 功能区块

  - 数字部分："3"，`36px`，`font-weight: 600`，`#1677FF`（次数充足时）或 `#FF8F1F`（次数不足时）

**C. 学习联动引导卡片（条件展示）**

- **图标**：视频/播放图标，`32px`，面性彩色，`linear-gradient(135deg, #FF8F1F, #FFBA3B)` 背景
- **标题**："今日次数已用完"，`15px`，`font-weight: 600`，`#1D2129`

#### 5.1.4 交互规则

| 触发条件 | 响应 |
|---------|------|

- 倒计时结束自动交卷
- 中途退出视为交卷
- 交卷后不可再次进入该阶段答题

#### 6.3.6 状态与异常

| 状态 | 表现 |
|------|------|
| 倒计时 < 5 分钟 | 倒计时文字变红 `#F53F3F`，每 1 秒闪烁一次 |
| 倒计时结束 | 自动触发交卷，按钮显示加载态 |
| 网络异常 | 本地缓存答题数据，提示"网络异常，答题已本地保存，请尽快恢复网络后提交" |

### 8.5 错误提示

| 场景 | 提示方式 |
|------|---------|

| 表单校验失败 | 输入框下方红色文字提示，`13px`，`#F53F3F` |
| 操作确认 | 底部弹窗或居中确认弹窗，标题 `16px`，内容 `14px`，按钮主/次级 |

### 8.6 页面跳转规范

| 跳转方式 | 应用场景 |
|---------|---------|
| Tab 切换 | 底部标签栏切换，无页面栈变化 |
| 普通跳转 | `wx.navigateTo`，保留当前页，如列表 → 详情 |
| 重定向 | `wx.redirectTo`，替换当前页，如登录 → 首页 |
| 返回 | `wx.navigateBack`，返回上一页，带交卷确认拦截 |
| 外部跳转 | `wx.navigateToMiniProgram` 或 WebView，跳转国家级平台 |
